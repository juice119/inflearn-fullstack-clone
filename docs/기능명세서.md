# 사용자

## 회원가입

- `email`, `password`, `nickname`, `passwordConfirm`으로 가입할 수 있다.
- `email`, `password`, `nickname`은 **필드 규칙**을 따른다.
- `passwordConfirm`은 `password`와 일치해야 한다.
  - 불일치 시 `비밀번호가 일치하지 않습니다.`를 표시한다.
- 검증 실패 시 에러 메시지를 표시한다.
  - 이메일 형식 오류: `올바른 이메일 형식이 아닙니다.`
  - 이메일 중복: `이미 사용 중인 이메일입니다.`
  - 닉네임 중복: `이미 사용 중인 닉네임입니다.`
  - 필드 규칙 위반: 해당 필드 아래에 규칙 위반 내용을 표시한다.
- 가입 성공 시 로그인 페이지로 이동하고 `회원가입에 성공했습니다.`를 표시한다.

## 로그인

- `email`, `password`로 로그인할 수 있다.
- `email`은 **필드 규칙 — `email`**의 형식 검증만 적용한다.
- `password`는 필수 입력이다. 형식 검증은 하지 않는다.
- 로그인 실패 시 에러 메시지를 표시한다.
  - 이메일 형식 오류: `올바른 이메일 형식이 아닙니다.`
  - 이메일 미존재: `이메일이 존재하지 않습니다. 이메일을 다시 확인해주세요.`
  - 비밀번호 불일치: `비밀번호가 올바르지 않습니다. 비밀번호를 다시 확인해주세요.`
- 로그인 성공 시 메인 페이지로 이동한다.

## 프로필 수정

- 로그인한 사용자만 수정할 수 있다.
- `nickname`, `password`, `image`, `bio`를 각각 또는 함께 수정할 수 있다.

### `nickname` 수정

- **필드 규칙 — `nickname`**을 따른다.
- 본인의 기존 닉네임과 동일한 값은 중복으로 보지 않는다.
- 닉네임 중복 시 `이미 사용 중인 닉네임입니다.`를 표시한다.

### `password` 수정

- `currentPassword`, `password`, `passwordConfirm`을 입력받는다.
- `currentPassword`는 현재 비밀번호와 일치해야 한다.
  - 불일치 시 `현재 비밀번호가 올바르지 않습니다.`를 표시한다.
- `password`는 **필드 규칙 — `password`**를 따른다.
- `passwordConfirm`은 `password`와 일치해야 한다.
  - 불일치 시 `비밀번호가 일치하지 않습니다.`를 표시한다.

### `image` 수정

- 프로필 이미지 URL을 입력하거나 파일을 업로드해 등록할 수 있다.
- 허용 형식: `jpg`, `png`, `gif`
- 최대 5MB

### `bio` 수정

- 자기소개 텍스트를 입력할 수 있다.
- 앞뒤 공백은 trim한다.

- 수정 성공 시 `프로필이 수정되었습니다.`를 표시한다.

## 필드 규칙

아래 규칙은 회원가입·프로필 수정에서 동일하게 적용한다.

### `email`

- 이메일 형식이어야 한다.
- 앞뒤 공백은 trim한다.
- 중복 불가. 대소문자를 구분한다. (`User@mail.com` ≠ `user@mail.com`)

### `password`

- 허용 문자: 영문·숫자·특수문자 (`!@#$%^&*()` 등, 공백 제외)
- 길이 6~16자 (영문 대소문자 구분)
- 영문 1자 이상, 숫자 1자 이상 포함
- 앞뒤 공백은 trim하지 않는다.

### `nickname`

- 허용 문자: 한글·영문·숫자 (영문 대소문자 구분)
- 한글 또는 영문 최소 1자 포함
- 길이 2~8자
- 앞뒤 공백은 trim한다.
- 중복 불가

# 카테고리

## 카테고리 목록

- 전체 카테고리 목록을 조회할 수 있다.
- 각 카테고리는 `name`, `slug`를 가진다.
- 사이트 헤더에서 카테고리를 선택하면 해당 카테고리의 강의 리스트로 이동한다.

---

# 강의

## 필드 규칙

### `title` (강의 제목)

- 필수 입력
- 앞뒤 공백은 trim한다.
- 길이 1~100자

### `description` (상세소개)

- HTML 형식으로 작성한다.
- 강의 상세 페이지·수업 설명에 노출된다.

### `thumbnailUrl` (커버 이미지)

- 허용 형식: `jpg`, `png`, `gif`
- 최대 5MB
- 권장 비율: 1200 × 781

### `status` (공개 상태)

- `DRAFT` (임시저장) · `PUBLISHED` (공개) 중 하나
- 생성 시 기본값은 `DRAFT`이다.

### `section.title`

- 필수 입력
- 앞뒤 공백은 trim한다.
- 길이 1~200자

### `lecture.title`

- 필수 입력
- 앞뒤 공백은 trim한다.
- 길이 1~200자

### `lecture.video`

- 강의당 영상 1개
- 허용 형식: `mp4`, `mkv`, `mov`
- 최대 300MB

## 강의 리스트 조회

- `PUBLISHED` 상태의 강의만 노출한다.
- 강의 카드에 `title`, `thumbnailUrl`, `discountPrice`, 지식공유자 `nickname`을 표시한다.
- 필터 및 검색을 지원한다.
  - `title`: 제목 부분 일치 검색 (대소문자 무시)
  - `categoryId`: 카테고리 필터
- 페이지네이션을 지원한다. (`skip`, `take` — 기본 `take` 30)
- 최신 생성 순으로 정렬한다.

## 강의 상세 페이지

- 강의 ID로 상세 정보를 조회할 수 있다.
- `PUBLISHED` 강의만 제한 없이 사용자에게 노출한다. `DRAFT` 강의는 생성한 지식공유자만 조회할 수 있다.
- 다음 정보를 표시한다.
  - 강의 정보: `title`, `description`, `thumbnailUrl`
  - 지식공유자: `nickname`
  - 커리큘럼: `section` · `lecture` 목록 (제목, 수업 시간, 미리보기 여부)
- 로그인한 사용자에게 수강 신청 버튼을 표시한다.
  - 이미 수강 중이면 `학습하기` 버튼을 표시한다.

## 강의 생성

- 로그인한 사용자만 강의를 생성할 수 있다.
- `title`을 입력해 강의를 생성한다.
- 생성 시 `status`는 `DRAFT`로 설정된다.
- `slug`는 `title`을 기반으로 자동 생성한다.
- 생성 성공 시 강의 편집 페이지(`/course/{id}/edit/course_info`)로 이동한다.

## 강의 수정

- 자신이 생성한 강의만 수정할 수 있다.
- 편집 화면은 4개 탭으로 구성한다.

### 강의 정보 (`course_info`)

- `title`를 수정할 수 있다.
- **필드 규칙**을 따른다.
- 저장 성공 시 `강의 정보가 수정되었습니다.`를 표시한다.

### 커리큘럼 (`curriculum`)

- `course` → `section` → `lecture` 3단 구조로 관리한다.
- 자신이 생성한 `course`만 커리큘럼을 수정할 수 있다.

#### `section`

- `title`을 입력해 섹션을 생성할 수 있다.
- 생성 시 목록 맨 아래에 추가되며 `order`가 자동 부여된다.
- 섹션 제목을 인라인으로 수정할 수 있다.
- 섹션을 삭제할 수 있다. (소프트 삭제)
- 섹션 간 순서를 변경할 수 있다.

#### `lecture`

- `title`을 입력해 섹션에 수업을 생성할 수 있다.
- 생성 시 해당 섹션 목록 맨 아래에 추가되며 `order`가 자동 부여된다.
- 수업을 삭제할 수 있다. (소프트 삭제)
- 수업 오른쪽 수정 버튼을 클릭해 수정할 수 있다.
  - `title` 수정
  - `description` 수정 (HTML 에디터)
  - 영상 1개 추가 및 변경 (`lecture.video` 규칙 적용)
- `isPreview`를 설정하면 비수강생도 해당 수업을 미리볼 수 있다.

### 상세소개 (`description-builder`)

- `description`을 HTML 에디터로 작성·수정할 수 있다.
- 저장 성공 시 `상세소개가 저장되었습니다.`를 표시한다.

### 커버 이미지 (`cover-image`)

- `thumbnailUrl`을 추가 및 변경할 수 있다.
- **필드 규칙 — `thumbnailUrl`**을 따른다.
- 파일 업로드 후 URL을 저장한다.
- 저장 성공 시 `커버 이미지가 저장되었습니다.`를 표시한다.

# 학습

## 수강 신청

- 로그인한 사용자만 수강 신청할 수 있다.
- 동일 강의에 중복 수강 신청할 수 없다.
- 수강 신청 성공 시 `수강 신청이 완료되었습니다.`를 표시한다.

## 학습 페이지

- 수강 중인 강의 또는 `isPreview` 수업만 접근할 수 있다.
- URL: `/course/{courseId}/learn/{lectureId}`
- 영상 플레이어로 수업 영상을 시청할 수 있다.
- 왼쪽에 커리큘럼 목록을 표시하고, 현재 수업을 하이라이트한다.
- 수업 `description`을 영상 아래에 표시한다.

## 이어보기

- 시청 진행률(`progress`)을 저장한다.
- 강의 상세·학습 페이지 진입 시 마지막으로 시청한 수업부터 이어볼 수 있다.
- 수업 시청 완료 시 `isCompleted`를 `true`로 기록한다.
- 강의 전체 진행률(완료 수업 수 / 전체 수업 수)을 표시한다.

# 지식공유자

## 지식공유자 허브

- 로그인한 사용자가 지식공유자 메뉴에 접근할 수 있다.
- 새 강의 만들기, 강의 관리 등 지식공유자 기능으로 이동할 수 있다.

## 강의 만들기

- 강의 생성 페이지로 이동한다.

## 강의 관리

- 로그인한 사용자가 **본인이 생성한 강의**만 목록에 표시한다.
- 각 강의에 `title`, `status`, 수강생 수, 생성일을 표시한다.
- 강의 행 클릭 시 해당 강의 편집 페이지로 이동한다.
- 강의 삭제를 지원한다.
  - 삭제 전 확인 다이얼로그를 표시한다.
  - 삭제 성공 시 `강의가 삭제되었습니다.`를 표시한다.
